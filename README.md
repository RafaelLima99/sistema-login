# Sobre o projeto

Sistema de login e cadastro de usuário feito em PHP puro, usando o padrão de arquitetura MVC.

# Tecnologias utilizadas
## Back end
- PHP
- Banco de dados: MySQL
## Front end
- HTML 5 / CSS 3 / JS 
- JQuery
- Bootstrap
# Como executar o projeto
Pré-requisitos: PHP >= 7
